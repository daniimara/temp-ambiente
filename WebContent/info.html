Introdução

A aplicação é destinada ao gerenciamento de temperatura de sensores instalados nos ambientes cadastrados na aplicação. Permitindo que a aplicação seja utilizada em diversos contextos, e customizada de acordo com as necessidades específicas do negócio.

Na imagem abaixo, a placa Arduino, conectada a um sensor de temperatura, envia as informações sobre a temperatura (Celsius) em um intervalo regular de tempo para o serviço de IoT Foundation no IBM Bluemix por meio do protocolo de comunicação do IoT - MQTT, e a temperatura do sensor é registrada no banco de dados da aplicação de acordo com o sensor cadastrado na aplicação.

     Fechar

Sempre que a temperatura do sensor é atualizada no banco de dados da aplicação, é realizado o cálculo médio da temperatura do ambiente, e informado na tela principal da aplicação junto aos sensores em alerta.

A temperatura do sensor correspondente a imagem, é atualizada no registro de Sensor Real da aplicação. Foram adicionados outros sensores simulados no serviço de IoT Foundation no IBM Bluemix.

Quando a temperatura do sensor não está no intervalo de temperatura minímo e máximo definido para o ambiente, as informações do sensor são exibidas na tela principal da aplicação como alerta.

Na imagem abaixo, a temperatura do Sensor Real é de 19 °C, e o intervalo de temperatura definido para o ambiente do sensor é de 20 °C a 30 °C, assim, o sensor é exibido como alerta na aplicação. Considerando todos os sensores que estão no ambiente Sala de Reunião 1 - 3o Andar, a temperatura média do ambiente corresponde a 36 °C, e este ambiente pertence ao local IBM Tutóia SP.

Informações Sobre Local

Voltar

A tela inicial de local, exibe todos os locais cadastrados:

  • Para cadastrar um novo local, selecione o botão "Novo" logo acima no canto direito da tela. Será necessário informar os seguintes campos:

    1. Campo Descrição: Este campo é obrigatório e alfanumérico.
    2. Campo Status: Este campo é obrigatório e será possivel selecionar as opções Ativo e Inativo.
    3. Selecione a opção "Salvar".

  • Para alterar um local, selecione um item da lista. Assim será possivel modificar os seguintes campos:

    1. Campo Descrição: Este campo é obrigatório e alfanumérico.
    2. Campo Status: Este campo é obrigatório e será possivel selecionar as opções Ativo e Inativo.
    3. Selecione a opção "Salvar".

  • Para excluir um local, selecione um item da lista.

    1. Selecione o botão "Excluir".
    2. Confirme que deseja realmente excluir o item escolhido.

Informações Sobre Ambiente

Voltar

A tela inicial de Ambiente, exibe todos os Ambientes cadastrados:

  • Para cadastrar um novo Ambiente, selecione o botão "Novo" logo acima no canto direito da tela. Será necessário informar os seguintes campos:

    1. Campo Descrição: Este campo é obrigatório e alfanumérico.
    2. Campo Status: Este campo é obrigatório e será possivel selecionar as opções Ativo e Inativo.
    3. Campo Local: Este campo é obrigatório e será possivel selecionar os locais cadastrados anteriormente.
    4. Campo Temperatura Mínima: Este campo é obrigatório e numérico.
    5. Campo Temperatura Média: Este campo corresponde a temperatura média do ambiente, ele será calculado automaticamente quando a temperatura do sensor for atualizada pelo serviço de IoT no IBM Bluemix.
    6. Campo Temperatura Máxima: Este campo é obrigatório e numérico.
    7. Selecione a opção "Salvar"

  • Para alterar um Ambiente, selecione um item da lista. Assim será possivel modificar os seguintes campos:

    1. Campo Descrição: Este campo é obrigatório e alfanumérico.
    2. Campo Status: Este campo é obrigatório e será possivel selecionar as opções Ativo e Inativo.
    3. Campo Local: Este campo é obrigatório e será possivel selecionar os locais cadastrados anteriormente.
    4. Campo Temperatura Mínima: Este campo é obrigatório e numérico.
    5. Campo Temperatura Média: Este campo corresponde a temperatura média do ambiente, ele será calculado automaticamente quando a temperatura do sensor for atualizada pelo serviço de IoT no IBM Bluemix.
    6. Campo Temperatura Máxima: Este campo é obrigatório e numérico.
    7. Selecione a opção "Salvar"

  • Para excluir um Ambiente, selecione um item da lista.

    1. Selecione o botão "Excluir".

Informações Sobre Sensor

Voltar

A tela inicial de Sensor, exibe todos os Sensores cadastrados:

  • Para cadastrar um novo Sensor, selecione o botão "Novo" logo acima no canto direito da tela. Será necessário informar os seguintes campos:

    1. Campo Descrição: Este campo é obrigatório e alfanumérico.
    2. Campo Id Sensor: Este campo é obrigatório e alfanumérico, com tamanho de 12 caracteres.
    3. Campo Status: Este campo é obrigatório e será possivel selecionar as opções Ativo e Inativo.
    4. Campo Ambiente: Este campo é obrigatório e será possivel selecionar os ambientes cadastrados anteriormente.
    5. Selecione a opção "Salvar".

  • Para alterar um Sensor, selecione um item da lista. Assim será possivel modificar os seguintes campos:

    1. Campo Descrição: Este campo é obrigatório e alfanumérico.
    2. Campo Id Sensor: Este campo é obrigatório e alfanumérico, com tamanho de 12 caracteres.
    3. Campo Status: Este campo é obrigatório e será possivel selecionar as opções Ativo e Inativo.
    4. Campo Ambiente: Este campo é obrigatório e será possivel selecionar os ambientes cadastrados anteriormente.
    5. Campo Temperatura: Este campo será preenchido automaticamente de acordo com a temperatura do sensor.
    6. Selecione a opção "Salvar".

  • Para excluir um Sensor, selecione um item da lista.

    1. Selecione o botão "Excluir".
    2. Confirme que deseja realmente excluir o item escolhido.
